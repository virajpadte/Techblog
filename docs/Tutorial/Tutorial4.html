Introduction

I recently came across a scenarios where I had to send key strokes or mouse movements to a remote host to which I connected via ssh.

Tutorial

Well after much tinkering and googling xdotool came to my rescue.

#installation: For Ubuntu, Debian or Linux Mint, you can just do:
$ sudo apt-get install xdotool
For Fedora, use yum command:
$ sudo yum install xdotool
Usage with remote host connection:
Example:
DISPLAY=:0 xdotool key Escape
