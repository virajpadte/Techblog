Introduction

Ever since I bought a MacBook I have been been wanted a good tutorial on setting up my Mac for all the software development work that I do. After a lot of searching online, I have come up with a collection of must read tutorial which any developer needs to follow to keep the development environment clean and free of any dependency issues. Some of the most important tools which will be discussed in these tutorials are HomeBrew, Xcode command line tools, dependency management for python and web development tools on Mac.

Tutorial

Follow this tutorial to configure your MacBook for using Homebrew. It is an excellent package manager which can be used for installing, updating and removing packages. Also managing different versions of same packages.

https://hackercodex.com/guide/mac-osx-mavericks-10.9-configuration/

Next, lets get to the core development tools. Since I love python, I have covered a tutorial for Installing python: https://hackercodex.com/guide/python-development-environment-on-mac-osx/


Next, using virtual environments in python. You will also learn how to use pip for installing and uninstalling python packages. This is especially important as installing python dependency could ruin your development environment. I have also found this to be really handy for easy migration of projects and providing a consolidated dependency list when you open source a developed software.

http://docs.python-guide.org/en/latest/dev/virtualenvs/
