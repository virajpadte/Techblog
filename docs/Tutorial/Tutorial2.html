Introduction

QT is one of the most popular GUI developement framworks out there. I deceided to experiment with it to develope a custom GUI for my laser engraver. Here is a set of instructions needed to get started with setting up a development environment for QT.

Procedure

Step 1: Get all the build essentials:
sudo apt-get install build-essential

Step 2: Install needed dependencies:
sudo apt-get install libxext-dev libfontconfig1-dev libfreetype6-dev libx11-dev libxcursor-dev libxfixes-dev libxft-dev libxi-dev libxrandr-dev libxrender-dev

Step 3: Download QT from which ever package version you want:
https://download.qt.io/archive/qt/

Step 4: Move the .tar.gz file to /tmp
sudo cp qt-everywhere-opensource-src-%VERSION%.tar.gz /tmp/

Step 5: Uncompress the archive
gunzip qt-everywhere-opensource-src-%VERSION%.tar.gz

Step 6: Unpack it
tar xvf qt-everywhere-opensource-src-%VERSION%.tar

Step 7: Configure and build the library
cd /tmp/qt-everywhere-opensource-src-%VERSION%
./configure

Step 8: Make and install
sudo make install

Step 9: Add path:
PATH=/usr/local/Trolltech/Qt-4.8.5/bin:$PATH
export PATH

Step 10: Download qt creator:
https://www.qt.io/download-open-source/

Step 11: Install the qt-creator using the run file
chmod +x qt-creator-opensource-linux-x86_64-version.run
./qt-creator-opensource-linux-x86_64-version.run

Step 12: Configure qt-creator
Select Tools > Options > Build & Run > Qt Versions > Add
Select /usr/local/Trolltech/Qt-version/bin/qmake
You have now configured qt with your qt-creator
